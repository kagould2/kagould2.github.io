Introduction

This page addresses some common misconceptions about animals often mistaken for bugs.

Common Misconceptions

List of Misconceptions

  • Bugs are insects.
  • All bugs are harmful.
  • Bugs can survive without air.

Description List of Misconceptions

Bugs are insects.
While bugs belong to the class Insecta, not all insects are bugs. Bugs are a specific type of insect characterized by a mouthpart modified for piercing and sucking.
All bugs are harmful.
Many bugs are beneficial, such as pollinators like bees and butterflies, and predators that control pest populations.
Bugs can survive without air.
Like all living organisms, bugs require oxygen for respiration. They cannot survive indefinitely without access to air.
