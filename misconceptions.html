Introduction

This page addresses some common misconceptions about animals often mistaken for bugs.

Common Misconceptions

Description List of Misconceptions

Bugs are insects.
While bugs belong to the class Insecta, not all insects are bugs. Bugs are a specific type of insect characterized by a mouthpart modified for piercing and sucking. Specifically, true bugs belong to the order Hemiptera.
